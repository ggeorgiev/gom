Presenting Boost.Gom

Boost.Gom is a library that provides a global objects management functionality to allow proper bootstrapping and shutdown of the applications. It creates mechanism for setting specifying initialization and uninitialization order of global objects solving multi-threading, dependency, mocking issues.

Note that Boost.Gom does not have for goal to promote or encoarage using of global objects. They should be concedered evil and avoided.

Users of this library should already be familiar with crc initialization, static object initialization, singlenton patterns, shared libraries loading and unloading mechanics.

The implementation makes use of crc initialization to construct the initialization list.

Purpose

The perpose of Boost.Gom is to create an relatively easy mechanism for initializng and unitializing global objects that with the existing c++ standards is a challange.
